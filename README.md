# mgt-ui

A modern React UI for [mgt](https://github.com/santhoshrox/mgt) — an enhanced Charcoal (gt) CLI wrapper for stacked PRs.

## Features

- **Dashboard** — Overview of your stacks, PRs, and quick CLI actions
- **Stack Visualization** — Interactive branch stack view with navigation (up/down/top/trunk)
- **PR Inbox** — Unified inbox with search, filters by status/author/CI/labels
- **AI Reviews** — Automatic code review with bug detection, suggestions, and feedback
- **Merge Queue** — Visual queue with CI job status, ETA, and position tracking
- **Insights** — Developer metrics: cycle time, review latency, deploy frequency, and activity charts

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- React Router v7
- Recharts (charts)
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```
